📊 Welcome to the SAS-Based MIS Application

Built for Business Analytics Coursework 2 (7CS512)

👋 Introduction: This web-based MIS platform allows users to upload datasets, apply dynamic filters, generate sampling files, and prepare configuration files for SAS analytics.

🛠️ How to Use This Tool

  1. Upload your dataset (CSV, Excel, or SAS7BDAT)
  2. Select the columns to include
  3. (Optional) Create new conditional columns
  4. Generate a 20K sample dataset
  5. Select SAS procedures and variables → Export config.csv
